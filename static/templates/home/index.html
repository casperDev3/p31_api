Що вміє платформа

Повноцінний бекенд для новинного порталу з сучасним стеком технологій

REST API

Повноцінний CRUD для статей, категорій, тегів і коментарів. JWT-аутентифікація та фільтрація через Django REST Framework.

GraphQL

Гнучкі запити через Graphene-Django. Отримуй лише ті дані, які потрібні — без зайвого навантаження на мережу.

WebSocket

Сповіщення в реальному часі через Django Channels та Daphne. Нові статті миттєво з'являються у підписників.

JWT Auth

Безпечна аутентифікація через SimpleJWT з підтримкою access та refresh токенів для REST і GraphQL запитів.

Адмін-панель

Красива адмін-панель на базі Unfold з дашбордом, графіками та зручним управлінням контентом.

Теги та категорії

Гнучка система організації контенту: статті, категорії, теги та коментарі з авторизацією авторів.
